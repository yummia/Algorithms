冒泡排序
  比较相邻的元素，如果第一个比第二个大，就交换他们两个。
  对每一对相邻元素做同样的工作，从开始第一对到结尾的最后一对。

  持续每次对越来越少的元素重复上面的步骤，直到没有任何一堆数字需要比较。
